# ModuMeta - Beta 

## 1. 프로젝트 소개
- 기술 스택 
  - Frontend : TypeScript, React, NextJS, Redux, Ant-Design, StyledComponents
  - Backend : Spring, mariaDB
  - Deployment : AWS , vercel
 
- 웹 서비스에 대한 개요
  - MetaverseStation은 회원들이 서로 자신의 메타버스 공간을 공유할 수 있는 커뮤니티입니다.


## 2. 주요 기능
- OAUTH로그인 (카카오,네이버,구글)
- 글쓰기, 수정, 삭제 
- 댓글, 대댓글 쓰기, 수정, 삭제
- 게시글 조회수, 좋아요
- 카테고리별 필터
- 페이지네이션
- 검색기능
- 이미지 슬라이더
- SSR
